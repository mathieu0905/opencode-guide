  ═══════ Deep Merge vs Shallow Merge ═══════

  Shallow Merge (浅合并, ❌ 不是 OpenCode 的方式):
    base:    { permission: { bash: "ask", edit: { "*": "allow" } } }
    overlay: { permission: { bash: "allow" } }
    结果:    { permission: { bash: "allow" } }
    → edit 配置丢失了!

  Deep Merge (深合并, ✓ OpenCode 的方式):
    base:    { permission: { bash: "ask", edit: { "*": "allow" } } }
    overlay: { permission: { bash: "allow" } }
    结果:    { permission: { bash: "allow", edit: { "*": "allow" } } }
    → edit 配置保留! bash 被覆盖

  ═══════ 完整示例 ═══════

  用户配置 (~/.config/opencode/opencode.json):

  │ {                                  │

  │     }                              │
  │   },                               │
  │   "permission": {                  │
  │     "bash": "ask",                 │
  │     "edit": { "*": "allow" }       │
  │   },                               │
  │   "default_model": "anthropic/     │
  │     claude-opus-4"                 │
  │ }                                  │
  └────────────────────────────────────┘
